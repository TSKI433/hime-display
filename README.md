## Quick usage

```bash
# Clone this repository
$ git clone https://github.com/yingpengsha/electron-vite-quick-start.git
# Go into the repository
$ cd electron-vite-quick-start
# Install dependencies
$ yarn install
# Run the app
$ yarn run start
```

## Build

```base
$ yarn run build
```
